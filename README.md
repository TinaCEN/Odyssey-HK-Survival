# Odyssey HK Survival

Real-world Hong Kong data integration, dynamic radar chart visualization.

## Project structure

- `index.html` – entry page
- `src/` – TypeScript game code
- `public/assets/` – game assets
- `scripts/` – helper scripts

## Notes

- Large binary assets are included under `public/assets/`.
- If you plan to publish this publicly and the asset set is large, consider using Git LFS.
